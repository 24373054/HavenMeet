## 🚀 快速使用

### 搜索技能
```bash
skillhub search <关键词>
# 示例：skillhub search calendar
```

### 安装技能
```bash
skillhub install <技能名>
# 示例：skillhub install filesystem-1-0-0
```

### 查看已安装技能
```bash
ls -1 /home/Matrix/.openclaw/workspace/skills/
```

### 更新技能
```bash
skillhub upgrade <技能名>
# 或更新所有：skillhub upgrade --all
```
